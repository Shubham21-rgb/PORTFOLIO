# Shubham's Portfolio

This is a professional portfolio for Shubham, an enthusiastic ML Engineer currently pursuing graduation from IIT Madras.

## Features
- Overview of Shubham's skills and academic background
- Professional and easy-to-navigate design
- Feedback section with a personal rating

## Setup
1. Clone the repository.
2. Open `index.html` in your web browser to view the portfolio.

## Usage
This portfolio can be used as a template for creating personal portfolios with a focus on showcasing skills and educational background.
